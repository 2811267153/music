### 使用 vue2 vue-cli3 创建的一个手机版webApp 目前的功能如下：
```
1. 手机登录  可使用 短信 和 账号密码 进行 登录

2. 获取 推荐歌曲  由于这个功能 需要登录  默认 我传的自己的账号

3. 获取 推荐MV

4. 搜索 歌曲  进行 播放 以及 热门搜索 支持 点击之后 直接跳转播放

5. 获取 推荐歌单 以及 歌单分类

6. 歌曲播放 

歌曲播放 目前仅支持 上一首 下一首 和 一起听 功能
```
